Kebijakan Privasi

GuruSuite menyimpan data yang diperlukan untuk membuat perangkat ajar, memproses order pembayaran, dan mengaktifkan paket pengguna.

Data yang Disimpan

  • Nama pengguna atau sekolah.
  • Email yang diisi pengguna.
  • Detail order pembayaran, status order, kode aktivasi, dan token lisensi.
  • Riwayat generate disimpan di browser pengguna melalui localStorage.

Penggunaan Data

Data digunakan untuk membuat invoice, mencocokkan pembayaran, mengirim kode aktivasi, dan menjaga akses paket berbayar.

Penyimpanan

Order pembayaran disimpan di server pada folder data yang dikonfigurasi. Admin perlu melakukan backup berkala melalui halaman admin.

Kontak

Untuk pertanyaan privasi atau penghapusan data order, hubungi email admin yang tercantum pada halaman pembayaran.
