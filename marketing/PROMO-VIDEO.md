# Video Promo GuruSuite

File utama:

- `gurusuite-promo-video.html`

Cara pakai:

1. Buka `marketing/gurusuite-promo-video.html` di browser.
2. Klik `Rekam video 20 detik`.
3. Setelah selesai, klik `Unduh video`.
4. Upload hasil `gurusuite-promo-video.webm` ke WhatsApp Status, Instagram Reels, TikTok, atau konversi dulu ke MP4 bila platform meminta MP4.

Cara otomatis untuk developer:

1. Pastikan Playwright tersedia di komputer.
2. Jalankan `node marketing/render-promo-video.js`.
3. File akan dibuat di `marketing/gurusuite-promo-video.webm`.

Naskah isi video:

- GuruSuite untuk guru SD, SMP, dan SMA.
- Pilih jenjang, kelas, semester, dan kurikulum.
- Generate modul ajar, silabus, ATP, program semester, dan perangkat pembelajaran.
- Pembayaran manual, konfirmasi via email, lalu aktifkan kode premium.
- Coba di `https://gurusuite.vercel.app`.

Catatan:

- Format video vertikal 9:16.
- Durasi sekitar 20 detik.
- Warna mengikuti identitas GuruSuite: kuning muda hangat dengan aksen hijau.
